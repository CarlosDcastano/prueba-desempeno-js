CRUDZASO:

Crudzaso is a collaborative task management project with two user profile types: user and administrator. Users cannot access admin views, and admins cannot access user views.

Users and administrators can create, edit, and delete tasks, filtering by factors such as priority, status, and category.

CHARACTERISTICS:

You can sign up with your name, email and password.
Your defoult role will be user, unless an admin change your role.
Once loged, you can create tasks, edit it and delete it.

As an admin:

The admin has access to all tasks on the bucket
Also, has a dashboard where can review the status of their team's tasks.

SYSTEM REQUIREMENTS:

Modern web browser (Chrome, Firefox, Safari, Edge).
Internet connection to download game files.
JavaScript enabled in the browser.
Device capable of running web applications (PC, Mac, tablets, etc.).

TOOLS USED:

HTML5
CSS3
JavaScript (ES6+)
